# Прогнозирование продаж

Веб-приложение для анализа и прогнозирования продаж с использованием NiceGUI.

## 📋 Описание

Приложение загружает данные о продажах из PostgreSQL и строит прогнозы на выбранный год с использованием различных моделей машинного обучения (Prophet, SARIMA, Holt-Winters).

## ✨ Функциональность

- Загрузка данных о продажах из базы данных PostgreSQL
- Отображение исторических данных по годам
- Прогнозирование на выбранный год тремя способами:
  - На основе месячных данных
  - На основе недельных данных
  - На основе ежедневных данных
- Прогресс-бар во время расчетов
- Округление сумм до сотен тысяч для удобства чтения
- Автоматическая перезагрузка при изменении кода

## 🛠 Технологии

- **NiceGUI 3.8.0** - интерфейс
- **PostgreSQL** - база данных
- **Pandas 3.0.1** - обработка данных
- **Prophet 1.3.0**, **statsmodels 0.14.6**, **scikit-learn 1.8.0** - модели прогнозирования

## 📦 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/forecasing.git
cd forecasing
```

2. Создайте виртуальное окружение
```bash
python -m venv .venv
source .venv/bin/activate  # для Linux/Mac
# или
.venv\Scripts\activate  # для Windows
```

3. Установите зависимости
```bash
pip install -r requirements.txt
```

4. Создайте файл .env для настроек базы данных:
```
DB_HOST=your_host
DB_NAME=your_db
DB_USER=your_user
DB_PASSWORD=your_password
DB_PORT=your_port
```

## 🚀 Запуск
```bash
python main.py
# или
python -m nicegui main.py
```

>Приложение будет доступно по адресу: http://localhost:8080

## 📊 Структура базы данных

Таблица kamtent.sales
- `pay_date` - дата платежа  
- `pay_summ` - сумма платежа  

Таблица kamtent.division_results 
- `year` - год  
- `total_amount_plan` - плановая сумма  
- `total_amount_summary` - фактическая сумма

## 🎯 Использование

1. Нажмите "Загрузить данные" - загрузятся исторические данные
2. Выберите год для прогноза
3. Выберите уровень агрегации:
    - По месяцам - прогноз на основе месячных данных
    - По неделям - прогноз на основе недельных данных
    - По дням - прогноз на основе ежедневных данных
4. Нажмите "Сделать прогноз" и дождитесь результатов
